한글과컴퓨터 사전의 다양한 기능과 특징

 

도구-사전 모음-한글과컴퓨터 사전

[한글과컴퓨터 사전]은 아무개 문자 검색 기능, 단어 자동 인식 기능, 찾은 단어 복습 기능 등 다양하고 편리한 기능들을 지원하므로, 원하는 단어를 쉽고 빠르게 찾을 수 있습니다.

참고

아무개 문자 검색

찾으려는 단어의 철자를 정확히 모르거나 특정 철자가 포함된 영어 단어를 찾을 때 아무개 문자(?,*)를 이용하여 단어를 검색합니다.

예를 들어, 단어 입력 상자에 *se?ed를 입력하고 <Enter>를 누르면 중간에 se가 들어가고 다음에 한 개의 문자열을 포함하면서 ed로 끝나는 모든 단어를 찾아 [아무개 문자 검색 결과] 대화 상자에서 목록을 보여 줍니다. 그 중에서 원하는 단어를 고를 수 있습니다.

입력 예 설명
* 모든 단어들을 찾습니다.
fr* fr-로 시작하는 모든 단어들을 찾습니다.
fr*ed fr-로 시작하고 ed로 끝나는 모든 단어들을 찾습니다.
*ed -ed로 끝나는 모든 단어들을 찾습니다.
st??? st로 시작하는 5개의 철자로 된 모든 단어들을 찾습니다.
st??ion st-로 시작하고 -ion으로 끝나는 7자의 모든 단어들을 찾습니다.
??ion -ion으로 끝나는 5개의 철자로 된 모든 단어들을 찾습니다.

찾은 단어 본문에 삽입하기

단어 입력 상자에 입력된 내용을 본문의 커서 위치에 삽입할 수 있습니다.

[한글과컴퓨터 사전]이 실행된 상태에서 <Ctrl+Enter>를 누르면 단어 입력 상자의 내용이 한/글의 본문에 곧바로 입력됩니다.

단어 자동 인식

한글과컴퓨터 사전이 실행된 상태에서, 검색하려는 단어에 마우스를 갖다 놓으면 해당 단어를 자동으로 찾아 줍니다. 단어 입력 상자에 일일이 단어를 입력할 필요가 없이 단어를 실시간으로 찾아 주는 매우 편리한 기능입니다. 단어 자동 인식 기능을 이용하려면 [환경 설정]에서 [단어 자동 인식]을 선택하십시오.

이전에 찾았던 단어 보기(History 기능)

지금까지 찾았던 단어들의 목록을 볼 수 있습니다, 단어 입력 상자에서 목록 단추를 누르면 지금까지 찾았던 단어들의 목록을 보여 줍니다. 이 목록에서 원하는 단어를 선택하면 그 단어의 뜻풀이를 볼 수 있습니다.

설명 상자에서 선택한 내용만 보기

단어를 검색한 후 설명 상자 안에서 마우스 오른쪽 단추를 누르면, "변화형, 의미, 숙어, 관용구, 예문, 파생어, 참고" 목록을 가진 빠른 메뉴가 나타납니다. 이 중에서 설명 상자에 적용할 보이기/감추기를 선택하십시오.

복습 기능

지금까지 찾았던 단어들의 목록은 [복습창] 탭에서 확인할 수 있습니다. 예를 들어, "wonder"라는 단어를 검색했다면 복습창에는 "wonder" 단어가 들어 있는 엣센스 영한사전, 영중사전도 함께 기록됩니다. 따라서 다음에 해당 단어를 다시 검색할 때, 검색하려는 사전에 있는 "wonder" 단어 목록을 선택하면 됩니다.

복습창에 있는 찾은 단어 목록은 한글과컴퓨터 사전을 끝내더라도 그대로 남아 있습니다.

복습창 빠른 메뉴

복습창에서 마우스 오른쪽 단추를 누르면 빠른 메뉴가 나타납니다.

  • 불러오기

    • 저장한 복습 단어를 다시 불러와서 사용할 수 있습니다. 복습창의 빠른 메뉴에서 [불러오기]를 선택하면, [복습 단어 불러오기] 대화 상자가 나타납니다. 이전에 저장한 파일(*.his)을 선택하고 [열기] 단추를 누릅니다.

    이미 복습창에 등록된 단어가 있다면 현재 등록된 단어가 복습창 목록에서 삭제되고 불러온 파일의 내용만 등록됩니다.

  • 지우기

    • 복습창의 목록에서 선택한 항목 하나만 지워집니다.

  • 모두 저장하기

    • [복습 단어 저장하기] 대화 상자가 나타납니다. 파일 경로와 파일 이름(*.his)을 입력해 복습 단어를 저장합니다.

  • 모두 지우기

    • 복습 단어를 저장할 것인지를 묻습니다. [저장]을 선택하면 [복습 단어 저장하기] 대화 상자가 나타납니다. 파일 경로와 파일 이름(*.his)을 입력해 복습 단어를 저장합니다.

크기 조절

화면에 나타나는 사전의 설명 상자의 크기가 너무 작거나 너무 커서 보기에 불편한 경우 크기를 쉽게 조절할 수 있습니다, 창틀에 마우스 포인터를 가져다 놓으면 마우스 포인터의 모양이 바뀝니다. 이 상태에서 마우스 왼쪽 단추를 누른 채로 마우스 끌기를 하면 마우스를 놓은 위치로 창의 크기가 조절됩니다.

설명 상자 보기 설정

단어의 뜻풀이를 보여주는 [설명 상자]에 마우스 포인터를 놓고 마우스 오른쪽 단추를 누르면 "사전 보기"를 설정할 수 있는 빠른 메뉴가 나타납니다. 여기에서 단어에 대한 "변화형, 의미, 숙어/관용구, 예문, 파생어, 참고" 등을 뜻풀이 상자에 보일지의 여부를 결정할 수 있습니다.
