DLS

종단선형 그리기 (Draw Longitudinal Section)

종단곡선을 그립니다.

작성중인 도면에 종단곡선을 그리며, 경사표시 및 K값이 계산되므로 종단설계시 활용할 수 있습니다.
지반선만 있으면 검토 목적으로 몇 개의 종단선형을 그려 종단면도를 만들 수 있습니다.

명령어

DLS

실행 방법

옵션값 설정 ▶ 곡선 길이 입력 ▶ 종단선형 계산

[지우고 다시] 버튼은 바뀐 곡선 길이 값으로 종단곡선을 다시 그립니다.

실행 옵션

하나의 종곡선

하나의 IP에 대한 종단 곡선만 그립니다.

연속된 종곡선

여러개의 IP에 해당하는 종단 선형을 만들고 종단면도를 만들기위해 사용합니다.

H 축척

종단 횡방향 축척

V 축척

종단 종방향 축척

곡선 길이

종단 곡선의 길이

계산 간격

지정한 간격으로 계산을 한 값으로 종곡선을 그립니다.

경사 문자 표시

선택할 경우 종단선형의 % 경사 값을 그립니다.

경사 문자 크기

경사를 표시하는 문자의 크기.

소수점

표시되는 경사 값의 소수점 자리수

표준 레이어 사용

선택할 경우 CALS 표준쳬계의 레이어 및 색상으로 종단면도를 작성

K=

종단곡선이 그려진 다음 계산된 종단곡선의 변화 비율

지우고 다시

종단곡선을 그리고 난 후에 사용할 수 있으며, 종단곡선의 시점, VIP, 종점을 다시 선택하지 않고 종단곡선을 다시 그립니다.

표기

Station, Number 체인 선택

측점 간격

종단의 측점 간격을 지정합니다. 측점 간격마다 지반세로선 및 계산값을 표시합니다.

시작 측점

종단의 측점 간격을 지정합니다.
시작 측점이 100일 경우 시작 측점은 STA는 0+100, NO는 5+0 부터 시작됩니다

문자 높이

종단 테이블의 체인 / 지반고 / 계획고 / 절성고 문자의 높이

표 소수점

종단 테이블에 표기되는 값의 소수점 자리수

K 값 표시

선택할 경우 종단곡선 변화비율을 종단상에 표시

테이블에 종단선형 표시

선택할 경우 종단 테이블 종단선형을 표시

지반 / 계획고

지반고 또는 계획고를 도면에 씁니다

종단선형 계산

연속적으로 종단 선형을 그리고 종단선형을 테이블에 표시
