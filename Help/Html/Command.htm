Magic for GstarCAD
 

명령어

Magic의 명령어는 명령어 입력창에서 직접 입력하거나 툴바에서 아이콘을 선택하여 실행할 수 있습니다.

명령어

  • MPL
    여러개의 도면파일을 일괄 출력
  • TCO
    문자의 내용을 다른 문자로 복사
  • TSW
    문자의 내용을 서로 교체
  • TEI
    문자 내의 숫자를 증가시켜 복사
  • CR
    문자를 사각형의 중심으로 배치
  • CTP
    원, 호를 폴리선으로 변환
  • PW
    폴리선의 두께를 변경
  • DLS
    종단곡선 그리기
