한글과컴퓨터 사전 도구 상자

 

도구-사전 모음-한글과컴퓨터 사전

[한글과컴퓨터 사전] 도구 상자에서 사전에 적용할 여러 가지 기능을 수행할 수 있습니다.

따라 하기

도구 상자 보이기/감추기

  1. [한글과컴퓨터 사전]의 제목 막대에서 한글과컴퓨터 사전 아이콘()을 누르거나 제목 막대에서 마우스 오른쪽 단추를 누릅니다.
  2. 빠른 메뉴가 나타나면 [도구 상자] 메뉴 앞의 선택 표시()를 확인한 다음 [도구 상자]를 선택합니다.
  3. [한글과컴퓨터 사전]에서 도구 상자가 사라집니다.
  4. 다시 빠른 메뉴를 실행하여 [도구 상제]를 선택하면 선택 표시가 나타나면서 [한글과컴퓨터 사전]에 도구 상자가 다시 나타납니다.

자세한 정보

도구 상자 아이콘 설명

아이콘 이름 설명
찾을 낱말 입력 상자 찾을 단어를 입력하고 <Enter>를 누릅니다.
입력 상자 옆의 목록 단추를 누르면 지금까지 찾았던 단어들의 목록을 보여 줍니다.
앞 낱말 현재 찾은 표제어에 대해서 앞 낱말의 표제어와 설명 상자의 내용을 보여 줍니다.
뒤 낱말 현재 찾은 표제어에 대해서 뒤 낱말의 표제어와 설명 상자의 내용을 보여 줍니다.
찾은 낱말 이전 단어 찾은 낱말 중에서 직전에 찾았던 표제어를 보여 줍니다.
찾은 낱말 다음 단어 찾은 낱말 중에서 직후에 찾았던 표제어를 보여 줍니다.
단어 목록창 보기/닫기 사전의 본문 내용만 보고 싶을 경우에 이 아이콘을 누르면 단어 목록창이 사라집니다.
다시 목록 창을 나타내려면 아이콘을 한 번 더 누릅니다.
본문 검색 단어의 뜻풀이 설명 상자에 보이는 내용 중에서 원하는 단어를 찾습니다.
[찾을 문자열]을 입력하고 [찾기] 단추를 눌러 원하는 내용을 찾습니다.
복사하기 본문의 내용을 한/글이나 다른 응용프로그램으로 복사합니다.
뜻풀이 설명 상자의 내용 중에서 복사하려는 부분을 블록으로 설정한 후 [복사하기] 아이콘을 누르면 해당 내용이 복사됩니다. 한/글 등 다른 응용프로그램에서 [붙이기]를 하면 복사한 사전의 내용이 그대로 붙여집니다.
단어 자동 인식 [단어 자동 인식] 기능을 사용할지 여부를 선택합니다.
단어 자동 인식 기능은 한글과컴퓨터 사전이 실행된 상태에서 화면에 나와 있는 아무 단어에나 마우스를 갖다 놓으면 해당 단어를 한글과컴퓨터 사전에서 자동으로 찾아 주는 매우 편리한 기능입니다.
환경 설정 한글과컴퓨터 사전을 좀 더 편리하게 사용하기 위해 사용 환경을 설정할 수 있도록 [환경 설정] 대화 상자가 나타납니다.
도움말 한글과컴퓨터 사전 도움말이 나타납니다.

참고

빠른 메뉴

[한글과컴퓨터 사전]의 제목 막대에서 한글과컴퓨터 사전 아이콘()을 누르거나 제목 막대에서 마우스 오른쪽 단추를 누르면 빠른 메뉴가 나타납니다.

  • 도구 상자

    • [도구 상자] 메뉴를 선택할 때마다 [한글과컴퓨터 사전] 도구 상자의 보이기/감추기 상태가 전환되면서 메뉴 이름 앞의 선택 표시()가 나타나거나 사라집니다.

  • 상황선

    • [상황선] 메뉴를 선택할 때마다 [한글과컴퓨터 사전] 상황선의 보이기/감추기 상태가 전환되면서 메뉴 이름 앞의 선택 표시()가 나타나거나 사라집니다.

  • 환경 설정

    • [환경 설정] 대화 상자에서 [한글과컴퓨터 사전]의 사용 환경을 설정할 수 있습니다.

  • 사전 닫기

    • [한글과컴퓨터 사전]을 종료합니다.
