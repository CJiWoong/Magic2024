문자표 <Ctrl+F10>
 

입력-문자표

글자판을 통하여 직접 입력할 수 없는 기호나 입력하기 불편한 각종 문자들을 사용자가 직접 선택하여 입력합니다.

한/글에서 제공하는 문자표는 유니코드 체계에 의한 "유니코드 문자표"와 한/글에서 지금까지 지원하던 "한글(HNC) 문자표"가 있습니다. 사용자가 쓰기 편한 문자표에서 원하는 문자를 찾으십시오.

따라 하기

  1. [입력-문자표]를 실행합니다.
  2. [문자표 입력] 대화 상자가 나타나면 [유니코드 문자표-문자 영역]에서 [여러 가지 기호]를 선택합니다.
  3. 문자 목록에서 코드 영역 2605의 "★"을 선택한 다음 [넣기] 단추를 누릅니다.
  4. 편집 화면의 현재 커서 위치에 "★"이 입력됩니다.

참고

여러 가지 문자를 한 번에 쓰기

문자표의 문자 목록에서 원하는 문자를 <SpaceBar>로 계속 누르면 여러 가지 문자를 편집 창의 현재 커서 위치에 한꺼번에 입력할 수 있습니다.

글쇠를 이용하여 커서 이동

[문자 선택] 상자에서는 마우스를 사용하거나 화살표 글쇠(<←>, <→>, <↑>, <↓>)로 원하는 문자를 선택할 수 있습니다.

글쇠 설명
Home/End [문자 영역] 목록에서는 가장 첫 문자 영역이나 마지막 문자 영역으로 커서를 옮깁니다.
[문자 선택] 상자에서는 가장 첫 글자나 마지막 글자로 커서를 옮깁니다.
PageUp/PageDown [문자 영역] 목록 또는 [문자 선택] 상자에서 내용을 이전 화면으로 넘기거나 다음 화면으로 넘겨 볼 수 있습니다.
Ctrl+↑/Ctrl+↓ [문자 영역] 목록에서 현재 선택된 문자 영역보다 이전 문자 영역이나 다음 문자 영역에 해당하는 문자 목록을 보여 줍니다.
Spacebar [문자 선택] 상자에서 SpaceBar를 누르면 현재 선택된 문자가 편집 창의 현재 커서 위치에 삽입되면서, 선택 상태는 한 칸 뒤의 문자로 옮겨집니다.

  [문자 선택] 상자에서 마우스 왼쪽 단추를 두번 눌러도 현재 선택된 문자가 편집 창의 현재 커서 위치에 바로 삽입됩니다. 이 때 [문자표 입력] 대화 상자는 닫히지 않기 때문에 연속적으로 문자를 삽입할 수 있습니다.

다른 대화 상자에서 입력은 <Ctrl+F10>

기호를 입력하는 것은 본문 편집 상태뿐만 아니라, 일부 기능의 대화 상자 입력 창에서도 필요합니다.

[편집-찾기]에서 기호를 입력하려면, 문자열 입력 상자 안에 커서를 두고 <Ctrl+F10>을 누르거나 마우스 오른쪽 단추를 눌러 나타나는 빠른 메뉴에서 [문자표]를 실행할 수 있습니다.

글자 모양

문자표에서 고른 문자가 편집 화면에 입력될 때는 커서가 있던 자리의 글자 크기와 속성을 그대로 따라갑니다.

문자표 목록 상자에서도 현재 커서에 설정된 [모양-글자 모양-글꼴]의 종류에 맞추어 보입니다.

최근 문자 기억

문자표에서는 가장 최근에 쓰인 문자가 선택되어 있는 상태로 대화 상자를 보여 줍니다.

다음에 같은 문자를 다시 쓰려면 [입력-문자표]를 실행한 다음 바로 <Enter>를 누르면 같은 문자를 쉽게 입력할 수 있습니다.

한/글 호환 영역

[유니코드 문자표]의 [문자 영역] 중에서 "이(YI) 글자마디, 이(YI) 부수, 한/글 호환 영역"의 문자들은 한/글 97의 문자표에만 들어있고 유니코드 문자표에는 없던 일부 히브리어 문자, 일부 한자, 원문자 조각, 키캡 등의 문자들을 유니코드 문자로 만들어 놓은 것입니다.

한중일 통합 확장 한자-B 영역 추가

한/글 2007은 유니코드 3.2를 지원하며 사용할 수 있는 한자가 4만여 자가 늘었습니다. 추가된 한자 확인은 [입력-문자표]를 실행한 다음 [유니코드 문자표] 탭에서 [문자 영역]을 "한중일 통합 확장 한자-B"를 선택하면 됩니다.

단축 키를 이용한 문자 입력

입력하고자 하는 문자의 코드 값을 알면 단축 키로 문자를 입력 할 수 있습니다.

편집 창에 직접 코드 값을 입력한 다음 <Alt+Shift+F10>을 누르면 코드 값에 해당하는 문자를 문서의 커서 위치에 삽입할 수 있습니다.

활용

입력된 문자의 코드 번호 확인 <F3>, <Ctrl+F10>

문자표를 이용하여 현재 문서에 쓰인 특정 문자의 코드 번호를 알아낼 수 있습니다.

코드 값을 알고자 하는 글자 앞에 커서를 놓고 <F3>을 누르고, 곧바로 <Ctrl+F10>을 누르면, 커서가 있던 위치의 글자에 해당하는 유니코드 16진수 코드 번호를 표시해 줍니다.
