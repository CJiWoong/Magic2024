문자표: 유니코드 문자표

 

입력-문자표

[문자표 입력] 대화 상자의 [유니코드 문자표] 탭에서는 키보드에 없는 유니코드 문자를 입력할 수 있습니다.

유니코드(Unicode) 문자는 컴퓨터 상에서 세계 각국의 언어를 통일된 방법으로 표현할 수 있게 제안된 국제적인 문자 코드 규약입니다.

따라 하기

  1. [입력-문자표]를 실행합니다.
  2. [문자표] 대화 상자가 나타나면 [유니코드 문자표] 탭을 선택합니다.
  3. [유니코드] 입력 상자에 "2605"를 입력합니다.
  4. [문자 영역]에서는 [여러 가지 기호]가 선택되어 있고, [문자 선택]에서는 "★"이 선택된 상태로 나타납니다.
  5. [넣기] 단추를 누르면 편집 창의 현재 커서 위치에 "★"이 삽입됩니다.

자세한 정보

[문자표 입력] 대화 상자의 [유니코드 문자표] 탭

문자 영역

원하는 문자가 속해 있는 영역을 선택합니다.

문자 선택

[문자 영역]에서 선택한 영역에 포함된 문자 목록을 보여 줍니다.

유니코드

[문자 선택] 상자에서 선택한 문자의 유니코드를 보여 줍니다. 원하는 문자의 16진수 코드 번호를 직접 입력하면 해당 문자를 목록 상자에 보여 줍니다.

최근 사용한 문자

최근 사용한 문자가 표시됩니다.

선택 문자 확대

[문자 선택] 상자에서 문자에 마우스를 갖다 대면 문자를 확대해서 보여 주는 기능입니다.

참고

글쇠를 이용하여 커서 이동

[문자 선택] 상자에서는 마우스를 사용하거나 화살표 글쇠(<←>, <→>, <↑>, <↓>)로 원하는 문자를 선택할 수 있습니다.

글쇠 설명
Home/End [문자 영역] 목록에서는 가장 첫 문자 영역이나 마지막 문자 영역으로 커서를 옮깁니다.
[문자 선택] 상자에서는 가장 첫 글자나 마지막 글자로 커서를 옮깁니다.
PageUp/PageDown [문자 영역] 목록 또는 [문자 선택] 상자에서 내용을 이전 화면으로 넘기거나 다음 화면으로 넘겨 볼 수 있습니다.
Ctrl+↑/Ctrl+↓ [문자 영역] 목록에서 현재 선택된 문자 영역보다 이전 문자 영역이나 다음 문자 영역에 해당하는 문자 목록을 보여 줍니다.
Spacebar [문자 선택] 상자에서 SpaceBar를 누르면 현재 선택된 문자가 편집 창의 현재 커서 위치에 삽입되면서, 선택 상태는 한 칸 뒤의 문자로 옮겨집니다.

  [문자 선택] 상자에서 마우스 왼쪽 단추를 두번 눌러도 현재 선택된 문자가 편집 창의 현재 커서 위치에 바로 삽입됩니다. 이 때 [문자표 입력] 대화 상자는 닫히지 않기 때문에 연속적으로 문자를 삽입할 수 있습니다.

글자 모양

문자표에서 고른 문자가 편집 화면에 입력될 때는 커서가 있던 자리의 글자 크기와 속성을 그대로 따라갑니다.

문자표 목록 상자에서도 현재 커서에 설정된 [모양-글자 모양-글꼴]의 종류에 맞추어 보입니다.

한/글 호환 영역

[유니코드 문자표]의 [문자 영역] 중에서 "이(YI) 글자마디, 이(YI) 부수, 한/글 호환 영역"의 문자들은 한/글 97의 문자표에만 들어있고 유니코드 문자표에는 없던 일부 히브리어 문자, 일부 한자, 원문자 조각, 키캡 등의 문자들을 유니코드 문자로 만들어 놓은 것입니다.

활용

입력된 문자의 코드 번호 확인 <F3>, <Ctrl+F10>

[문자표]를 이용하여 현재 문서에 쓰인 특정 문자의 코드 번호를 알아낼 수 있습니다.

코드 값을 알고자 하는 글자 앞에 커서를 놓고 <F3>을 누르고, 곧바로 <Ctrl+F10>을 누르면, 커서가 있던 위치의 글자에 해당하는 유니코드 16진수 코드 번호를 표시해 줍니다.
