MPL

다중 플롯 (Multi PLot)

여러개의 도면 파일을 옵션을 설정하여 한번에 출력합니다.

도면파일별로 플롯 스타일을 지정할 수 있습니다

하나의 도면파일 내에 여러개의 블록, Xef된 Form이 있는 경우 각기 따로 출력을 할 수 있습니다.

명령어

MPL

실행 방법

출력할 도면파일을 목록에 추가한 다음 옵션을 설정하고 미리보기를 하여 내용을 확인한 후 출력을 합니다.
플로터 / 용지크기 / 스타일 선택 ▶ 출력할 도면 추가 ▶ 플롯 영역 설정 ▶ 축척 및 기타 옵션 설정 ▶ 출력

실행 옵션

변경시 목록 전체 변경

선택되어 있으면 플롯 스타일 변경될 경우 도면목록의 스타일이 전부 변경됩니다.

도면 추가

도면 파일을 선택하여 목록에 추가합니다.

폴더 추가

폴더를 선택하여 폴더 내에 있는 도면 파일들을 목록에 추가합니다.

활성 도면

현재 CAD에서 활성화 된 도면을 목록에 추가합니다.

열린 도면

현재 CAD에서 열린 모든 도면을 목록에 추가합니다.

선택 열기

선택한 목록의 도면파일을 CAD에서 엽니다.

선택 삭제

선택한 도면파일을 목록에서 삭제합니다.

전체 삭제

모든 목록을 삭제합니다.

목록 저장

목록의 내용을 Text 파일로 저장합니다.

목록 열기

Text 파일로 저장된 목록을 불러옵니다.

목록 정렬

목록을 오름차순/내림차순으로 정렬합니다.

플롯 영역 설정

Form : 출력 범위를 Form의 크기로 지정합니다. Form은 하나로 지정하든지 아니면 출력할 때 마다 Form을 지정할 수 있습니다.

Extents : 출력범위를 도면 내의 모든 객체의 크기로 지정합니다.

Limits : 출력범위를 도면 내에서 지정한 Limit로 지정합니다.

Layout : 출력범위를 모든 Layout 또는 특정 Layout으로 지정합니다.

도면에 저장된 설정대로 : 출력범위를 도면에 저장된 값으로 지정합니다.

플롯 대산 설정

활성된 도면 : 출력대상을 도면파일에 저장된 도면으로 지정합니다.

Model : 출력대상을 Model에 그려진 도형으로 지정합니다.

모든 Layout : 출력대상을 모든 Layout에 그려진 도형으로 지정합니다.

Model + 모든 Layout : 출력대상을 Model과 모든 Layout에 그려진 도형으로 지정합니다.

특정 Layout : 출력대상을 지정한 이름의 Layout에 그려진 도형으로 지정합니다.

용지에 맞춤

출력 용지에 맞게 크기를 조절하여 출력합니다.

자동 맞춤 (A시리즈 용지)

용지가 A1, A2, A3, A4로 선택되었을 경우 자동 축척으로 출력을 합니다.
