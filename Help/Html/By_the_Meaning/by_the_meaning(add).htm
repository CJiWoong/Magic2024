한자 새김 등록

 

입력-한자 입력-한자 새김 입력

[한자 새김 입력 사전]에는 약 4,100여 한자의 새김이 기본으로 등록되어 있으며, 사용자가 자주 쓰는 한자의 새김을 추가로 등록하여 쓸 수 있습니다.

따라 하기

  1. 한자 새김을 등록하기 위하여 [입력-한자 입력-한자 새김 입력]을 실행합니다.
  2. [한자 새김 입력] 대화 상자 아래쪽의 [한자 새김 등록] 단추를 누릅니다.
  3. [한자 새김 등록] 대화 상자가 나타나면 [사용자 사전] 탭을 선택한 다음, [한자 새김 추가하기]를 실행합니다.
  4. [한자 새김 추가하기] 대화 상자가 나타나면 [새김] 입력 상자에 등록할 한자의 새김(뜻과 음)을 빈칸 없이 이어서 입력합니다.
  5. [한자] 입력 상자로 커서를 옮기면 [새김]에 입력한 한자의 음이 자동으로 입력되어 나타납니다.
  6. [새김] 입력 상자에서 한자의 음 바로 뒤에 커서를 놓고 <F9>를 눌러 [한자로 바꾸기]를 실행합니다.
  7. [한자로 바꾸기] 대화 상자가 나타나면 [한자 목록]에서 알맞은 한자를 선택한 다음 [바꾸기] 단추를 누릅니다.
  8. [한자 새김 추가하기] 대화 상자의 [한자] 입력 상자에 한자가 올바르게 입력되면 [설정] 단추를 누릅니다.

자세한 정보

음을 한자로 바꾸기

[한자 새김 추가하기] 대화 상자의 [한자] 입력 상자에 입력한 한글 음에 해당하는 한자로 변환하기 위해서는 <F9>를 눌러 [한자로 바꾸기]를 실행하거나 마우스의 오른쪽 단추를 누르면 나타나는 빠른 메뉴에서 [한자로 바꾸기]를 선택합니다.

사용자 사전 지우기

한자 새김 사용자 사전에서 잘못 등록되었거나 필요 없는 것을 지우려면, 사용자 사전 목록에서 지우고자 하는 한자 새김을 선택한 다음 [한자 새김 지우기]를 실행합니다.

사용자 사전 파일 관리

사용자가 추가로 등록한 한자를 다른 시스템으로 옮겨 사용하거나 따로 보관해 둘 수 있습니다. [한자 새김 입력 사전]의 [사용자 사전] 탭에서 [한자 새김 사용자 사전 저장]을 실행하여 별도의 파일로 저장하십시오. 사용자 사전의 파일 형식은 "한자 새김 저장 파일(*.hjm)"입니다.

[한자 새김 사용자 사전 불러오기]를 실행하면, 따로 저장되어 있는 한자 새김 사용자 사전을 현재 사용하고 있는 한/글 프로그램에서 불러 쓸 수 있습니다.
