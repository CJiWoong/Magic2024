외국 문자 입력

입력-문자표

모두 보이기

글자판에서 직접 입력할 수 없는 유럽 여러 나라에서 사용하는 알파벳 문자들을 <Ctrl> 글쇠와 함께 두 개의 일반 영문 글자를 이용하여 손쉽게 본문에 입력할 수 있습니다.

따라 하기

글쇠 조합을 이용하여 입력하기
  1. [도구-글자판-글자판 바꾸기]를 실행합니다.
  2. [입력기 환경 설정] 대화 상자가 나타나면 [글자판 바꾸기] 탭을 선택합니다.
  3. [현재 글자판] 목록에서 [영어], [쿼티]를 차례로 선택하고 [설정] 단추를 누릅니다.
  4. 편집 화면에서 <Ctrl+`>를 입력한 후 곧바로 "a"를 입력합니다.
  5. 본문에 à(악상 떼기)가 입력됩니다.
문자표에서 입력하기
  1. [입력-문자표]를 실행합니다.
  2. [문자표 선택] 대화 상자가 나타나면 [유니코드 문자표] 탭을 선택합니다.
  3. [문자 영역] 목록에서 "라틴"을 선택합니다.
  4. [문자 선택] 목록 상자에서 "à"를 선택하고 [넣기]를 누릅니다.
  5. 본문에 à(악상 떼기)가 입력됩니다.
