환경 설정
 

도구-사전 모음-한글과컴퓨터 사전

모두 보이기

사전의 보기 상태와 글꼴 설정, 검색할 사전 구성 등 한글과컴퓨터 사전의 사용 환경을 설정합니다.

따라 하기

  1. [한글과컴퓨터 사전] 도구 상자의 [환경 설정] 아이콘을 누릅니다.
  2. [환경 설정] 대화 상자가 나타나면 [기본 환경] 탭에서 사전을 편리하게 사용할 수 있는 기능들의 사용 여부를 선택합니다.
  3. [글꼴 설정] 탭에서 글꼴은 "한컴돋움"을, 글자 크기는 "11"을 선택합니다.
  4. [본문 상태] 탭에서 뜻풀이 설명 상자에 표시되는 사전 내용들 각각에 대하여 글자 속성과 글자색을 설정합니다.
  5. [확인] 단추를 누릅니다.

자세한 정보

[환경 설정] 대화 상자

[기본 환경] 탭

검색

  • 단어 자동 인식
    [단어 자동 인식] 기능을 사용할지의 여부를 결정합니다. 단어 자동 인식 기능은 한글과컴퓨터 사전이 실행된 상태에서 화면에 나와 있는 아무 단어에나 마우스를 갖다 놓으면 해당 단어를 한글과컴퓨터 사전에서 자동으로 찾아 주는 매우 편리한 기능입니다.
  • 모든 프로그램 위에
    한글과컴퓨터 사전을 현재 실행되고 있는 프로그램의 가장 위에 위치시킬지의 여부를 결정합니다.
  • 기본형 찾기
    과거형, 현재 진행형, 복수형 등의 단어를 검색했을 때, 만약 그 단어들이 사전에 없다면 해당 단어들의 기본형을 찾아 줄지의 여부를 선택합니다.
  • 순서대로 찾기
    찾을 낱말 입력 상자에 한 글자씩 입력할 때마다 해당되는 단어들을 차례대로 보여 줄 것인지 여부를 선택합니다. 철자를 잘 모르는 단어를 검색할 때 이용하면 편리합니다.
  • 불규칙 동사/비교급
    단어를 검색했을 때, 불규칙 동사표나 비교급을 단어 설명 상자의 아랫부분에 따로 보여 줄지의 여부를 결정합니다.

보기

  • 도구 상자
    한글과컴퓨터 사전 도구 상자를 보이게 할 것인지 감출 것인지 선택합니다.
  • 상황선
    한글과컴퓨터 사전창의 아래 부분에 상황선을 보이게 할 것인지 감출 것인지 선택합니다.
[글꼴 설정] 탭

사전의 전체 글꼴과 크기를 변경합니다.

[본문 상태] 탭

사전에 표시되는 표제어, 발음, 일반 문자 등 각각의 항목에 대해 서로 다른 글자 속성과 글자 색상을 지정할 수 있습니다. 각각의 색상을 변경함으로써 각 내용의 구분을 눈에 잘 띄게 할 수 있습니다.

[사전 설정] 탭

"영중사전, 일한사전, 영한엣센스, 한일사전, 한영엣센스, 중영사전, 국어사전, 한글유의어/반의어사전, 영어유의어/반의어사전"의 한글과컴퓨터 사전 중에서 찾고자 하는 단어를 검색할 사전 종류와 순서를 지정합니다.

"사전 구성" 목록에서 검색하고자 하는 사전을 선택하고 [사전 목록에 추가하기] 단추를 눌러 "검색할 사전" 목록에 추가하십시오.

검색할 사전의 순서를 정하기 위해서, 검색할 사전을 선택하고 위/아래 이동 단추를 눌러 검색 순서를 조절하십시오.

참고

도구 상자 보이기/감추기

한글과컴퓨터 사전 도구 상자를 감춘 상태에서 다시 도구 상자를 보이게 하려면, [한글과컴퓨터 사전]의 제목 막대 앞쪽에 있는 프로그램 아이콘을 누르십시오. 또는 제목 막대에 마우스 포인터를 놓고 마우스 오른쪽 단추를 눌러 빠른 메뉴를 부릅니다. '도구 상자 보이기/감추기, 상황선 보이기/감추기, 환경 설정, 사전 닫기' 등의 메뉴가 펼쳐집니다.

주의

윈도우 95에서 사전 보기

한글과컴퓨터 사전의 모든 글자는 유니코드 폰트를 사용하기 때문에 윈도우 95에서는 글자가 제대로 보이지 않습니다. 이럴 때에는 [환경 설정-글꼴 설정] 탭의 [글꼴]을 "한컴바탕"이나 "한컴돋움"으로 설정해야 모든 글자들이 제대로 보입니다.
