한자 단어 사전

 

입력-한자 입력-한자로 바꾸기

모두 보이기

한/글에서 기본으로 제공하는 한자 단어 사전에는 16만 개 이상의 한자 단어가 등록되어 있습니다.

[한자로 바꾸기]를 실행했을 때, 자주 쓰는 한자 단어가 사전에 등록되어 있지 않으면 사용자가 직접 추가로 등록하고, 등록한 한자 단어 사전은 별도의 파일로 저장할 수 있습니다. 저장된 사전 파일은 다른 시스템에서도 불러 쓸 수 있습니다.

회사 이름이나 상표명과 같이 기본으로 제공되지 않는 한자 단어는 사용자가 직접 추가하여 사용하십시오.

따라 하기

새로운 단어 등록하기
  1. "服事"를 입력하기 위하여 한글로 "복사"라고 입력합니다.
  2. "복사" 단어 뒤에 커서를 놓고 <F9>를 눌러 [입력-한자 입력-한자로 바꾸기]를 실행합니다.
  3. [한자로 바꾸기] 대화 상자가 나타나면 기본 한자 단어 사전에 "服事"라는 한자 단어가 없으므로 [한자 단어 등록] 단추를 누릅니다.
  4. [한자 단어 등록] 대화 상자가 나타나면 [등록할 한자 단어-한글], [등록할 한자 단어-한자]에 각각 "복사"라고 입력되어 있는 것을 확인합니다.
  5. [바꾸기-한 글자씩 연속 바꾸기]를 선택하고 [한자로] 단추를 누릅니다.
  6. [한자로 바꾸기] 대화 상자의 [한자 목록] 상자에서 "服"를 선택하고 [바꾸기] 단추,"事"를 선택하고 [바꾸기] 단추를 누릅니다.
  7. [한자 단어 등록] 대화 상자의 [등록할 한자 단어-한자]에 "服事"가 올바르게 나타나면 [등록] 단추를 누릅니다.
  8. [한자로 바꾸기] 대화 상자의 [한자 목록] 상자에 "服事"가 올바르게 등록된 것을 확인한 다음 [바꾸기] 또는 [닫기]를 누릅니다.
  9. 이제부터는 "복사"를 입력하고 <F9>를 누르면, 한자 단어 사전에 "服事"가 나타납니다.
등록한 단어 지우기

사용자가 한자 단어 사전에 등록한 단어가 잘못되었거나, 더 이상 필요하지 않을 때에는 그 단어를 지울 수 있습니다.

  1. 한자 단어 사전에 직접 등록한 단어 중 한자 단어 사전에서 지울 단어를 입력합니다.
  2. 해당 단어 뒤에 커서를 놓고 <F9>를 누릅니다.
  3. [한자로 바꾸기] 대화 상자의 [한자 목록] 상자에서 지우려는 한자 단어를 선택한 다음 <Delete> 또는 [단어 지우기] 아이콘을 누릅니다.
  4. 한자 단어 사전에서 해당 단어가 삭제됩니다.

참고

기본 사용자 한자 단어 사전

등록된 한자 단어는 사용자 한자 사전(hjuser6.dic)에 자동으로 추가됩니다.

기본으로 저장되는 사용자 한자 사전은 개인 데이터 폴더 아래 \Hnc\User\Shared\Dics 폴더에 저장됩니다.

개인 데이터 폴더의 위치는 [도움말-한글과컴퓨터 한/글 2007 정보-폴더 정보]에서 확인할 수 있습니다.

글자 수 제한

한자 단어로 등록할 수 있는 글자 수는 한 단어에 2~12자이며, 모든 글자는 반드시 붙여서 써야 합니다.

단어 수 제한

한자 단어 사전에 추가할 수 있는 단어의 수에는 제한이 없습니다. 사용자 시스템의 하드 디스크 공간이 허락하는 한도까지 사전을 등록할 수 있습니다.

활용

사용자 등록 한자 사전 보관하기(backup)

사용자의 시스템 운영 체제나 한/글 프로그램을 지우고 다시 설치하는 등의 작업을 할 때에는 추가로 등록하여 쓰던 사용자 한자 단어 사전을 사용자가 직접 따로 보관할 필요가 있습니다. 이런 경우 사용자 한자 단어 사전 파일을 만들어두면 다른 환경에서 불러와 똑같이 사용하면 유용합니다.

자세한 내용은 사용자 한자 단어 사전 보관 부분을 참조하시기 바랍니다.

관련 기능

한자로 바꾸기, 한자 부수/총획수, 한자 새김 입력, 사용자 한자 단어 사전 백업, 한자 단어 한꺼번에 등록
