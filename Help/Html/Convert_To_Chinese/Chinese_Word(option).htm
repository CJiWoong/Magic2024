선택 사항

 

입력-한자 입력-한자로 바꾸기

한글로 입력한 단어를 한자로 변경해 주는 기능인 [한자로 바꾸기]를 좀 더 편리하게 사용할 수 있도록 도와주는 몇몇 기능을 설정할 수 있습니다.

따라 하기

  1. 한자로 바꿀 낱말의 한글 발음을 입력합니다. 여기서는 "국가"를 입력합니다.
  2. "국가"를 블록으로 설정한 다음 [입력-한자 입력-한자로 바꾸기]를 실행하거나 단축 키 <F9>를 누릅니다.
  3. [한자로 바꾸기] 대화 상자에서 [선택 사항] 단추를 누릅니다.
  4. [선택 사항] 대화 상자가 나타나면 [선택 문자 확대]를 선택하고 [설정]을 누릅니다.
  5. [한자로 바꾸기] 대화 상자의 [한자 목록] 상자에서 "國家" 글자에 마우스를 가까이 대면 해당 글자를 확대하여 보여 줍니다.
  6. "國家"를 선택한 다음 [바꾸기]를 누릅니다.
  7. 본문에서 한글로 입력된 "국가"가 "國家"로 바뀝니다.

자세한 정보

선택 사항

  • 선택 문자 확대
    한자 목록에서 사용자가 선택한 한자를 알아보기 쉽게 하기 위해서 크게 확대하여 보여 줍니다. 만일 선택 문자를 확대하는 것이 오히려 주변의 다른 글자를 가려서 불편하다면 이 항목을 끄십시오.
  • 맨 앞으로
    가장 최근 사용한 한자를 한자 목록에서 맨 앞에 위치하도록 합니다.
  • 조사 건너뛰기
    조사가 포함된 단어를 한자로 바꿀 때 조사는 건너뛰고 한자로 바꾸기를 실행합니다.

사용자 파일

  • 파일 이름
    사용자가 한자를 등록했을 때 등록한 파일 이름을 표시하여 줍니다.
    파일의 자세한 위치는 한/글을 실행한 다음 [도움말-한글과컴퓨터 한글 2007 정보-폴더 정보-개인 데이터 폴더]에서 확인합니다.

관련 기능

한자로 바꾸기, 한자 부수/총획수, 한자 새김 입력, 한자 단어 사전 만들기
