사용자 한자 단어 사전 백업
 

입력-한자 입력-한자로 바꾸기

모두 보이기

사용자의 시스템 운영 체제나 한/글 프로그램을 지우고 다시 설치하는 등의 작업을 할 때에는 추가로 등록하여 쓰던 사용자 한자 단어 사전을 사용자가 직접 따로 보관할 필요가 있습니다. 이런 경우 사용자 한자 단어 사전을 만들어두었다가 다른 환경에서 백업 파일을 불러와 사용하면 유용합니다.

따라 하기

사용자 한자 단어 사전 저장하기
  1. [입력-한자 입력-한자로 바꾸기]를 실행합니다.
  2. [한자로 바꾸기] 대화 상자가 나타나면 [사용자 한자 사전 저장하기] 아이콘을 누릅니다.
  3. [사용자 한자 사전 저장하기] 대화 상자가 나타나면 위치와 파일 이름을 지정한 다음 [저장] 단추를 누릅니다.
    이 때, [파일 형식]은 "사용자 한자 사전 저장 파일(*.dic)" 형식으로 자동 지정됩니다.
  4. [한자로 바꾸기] 대화 상자에서 [닫기]를 누릅니다.
사용자 한자 단어 사전 불러오기
  1. [입력-한자 입력-한자로 바꾸기]를 실행합니다.
  2. [한자로 바꾸기] 대화 상자가 나타나면 [사용자 한자 사전 불러오기] 아이콘을 누릅니다.
  3. [사용자 한자 사전 불러오기] 대화 상자가 나타나면 "사용자 한자 사전 저장 파일(*.dic)"이 저장된 위치를 찾아 파일을 선택한 다음 [저장] 단추를 누릅니다.
  4. [한자로 바꾸기] 대화 상자에서 [닫기] 단추를 누릅니다.
  5. 사용자 한자 사전에 추가된 모든 단어에 대해 [한자로 바꾸기]를 수행할 수 있습니다.

참고

사용자 정의 데이터 백업 기능

  • [도구-환경 설정-파일 위치]에서 [사용자 정의 데이터 불러오기], [사용자 정의 데이터 저장하기] 기능을 이용하면 한/글을 사용하면서 사용자가 추가/저장해 두었던 데이터를 다른 시스템에서 사용하거나 필요할 때 다시 불러 쓸 수 있습니다.
  • 백업 가능한 사용자 정의 데이터의 종류에는 "매크로, 상용구, 한자 단어 사전, 한자 새김 입력 사전, 빠른 교정, 입력 자동 명령 데이터, 맞춤법 개인 사전, 맞춤법 등록 기호, 로마자 등록 데이터" 등이 있습니다.
  • 사용자 정의 데이터를 저장할 때에는 모든 사용자 정의 데이터를 한꺼번에 저장하고, 불러올 때에도 모든 데이터를 한꺼번에 불러옵니다. 사용자가 원하는 데이터만 선택하여 저장하거나 불러 쓰려면 데이터별로 별도의 백업 기능을 이용해야 합니다.
